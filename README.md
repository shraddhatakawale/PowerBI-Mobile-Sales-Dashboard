📊 Mobile Sales Data Dashboard

Project Overview

This project presents a Mobile Sales Dashboard created using Microsoft Power BI.
The dashboard analyzes mobile phone sales data across different cities, brands, and payment methods to provide insights into sales performance and customer behavior.

Key Insights

- Total Sales: 769.20M
- Total Quantity Sold: 19K
- Total Transactions: 4K
- Average Price: 40.11K

Dashboard Features

- 📅 Monthly Sales Filter (January – December)
- 💳 Payment Method Analysis (Cash, Credit Card, Debit Card, UPI)
- 📍 City-wise Sales Performance
- 📱 Brand-wise Mobile Model Distribution
- 📈 Yearly and Quarterly Sales Trend
- ⭐ Customer Rating Analysis

Tools Used

- Microsoft Power BI
- Data Cleaning and Visualization

Project Files

- "Mobile_Sales_Dashboard.pbix" – Power BI dashboard file
- "dataset.xlsx" – Dataset used for analysis
- "dashboard.png" – Screenshot of the dashboard

Insights from the Dashboard

- Mumbai has the highest mobile sales among the cities.
- Apple and Samsung dominate the mobile model distribution.
- Most transactions fall under 4 and 5 customer ratings.
